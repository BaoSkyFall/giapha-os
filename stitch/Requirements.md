Role:
You are an expert UI/UX designer creating a modern web application interface.

Task:
Design a Story-First web application UI for a Vietnamese family genealogy platform called "Tộc Phạm Phú".

Context:
The application allows users to explore and manage a Vietnamese family tree (gia phả), manage family members, track family events, and search kinship relationships.

Design style: Modern Flat Design with Subtle Layers.

Color palette:
Primary: #B31D1D (Heritage Red)
Accent: #E8B931 (Prosperity Gold)
Secondary: #227C44 (Jade Green)
Text: #3E2723 (Altar Wood)
Accent Pink: #D46A84
Background: warm rice paper off-white (#F7F3EA).

Typography:
Headings: Playfair Display
Body: Inter

Example components:
Family member cards look like paper cards with soft shadows and gold borders. Each node shows avatar, name, birth year and lineage branch.

Instruction:
Create a clean web application layout including:

Header:
Logo “Tộc Phạm Phú”, navigation (Gia phả, Lịch sử dòng họ, Thành viên, Sự kiện, Tra cứu quan hệ), and a primary CTA button “Khám phá cây gia phả”.

Hero section:
Large headline “Lưu giữ lịch sử – Kết nối hậu duệ”, short subheadline describing the genealogy platform, two buttons “Xem cây gia phả” and “Thêm thành viên”, background illustration of a family tree with subtle traditional Vietnamese patterns.

Main content:
Interactive genealogy tree viewer allowing users to toggle between horizontal tree and vertical tree layout, zoom controls, and member search.

Additional sections:
Ancestor highlight cards, family event timeline, and a kinship relationship search interface.

Footer:
Family information, contact, and copyright.

Use consistent card sizes, subtle shadows, soft hover effects, and generous spacing to keep the interface elegant and easy to read.
